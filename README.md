# Kickstarter Data Visualization

This project contains analysis and visualization of Kickstarter project data.

## Files

- `viz.ipynb` - Jupyter notebook with data analysis and visualizations
- `Kickstarter Projects_639378_Aug 2025 Updated.csv` - Kickstarter dataset

## Usage

Open `viz.ipynb` in Jupyter Notebook or JupyterLab to view the analysis.

## Requirements

See the notebook for required dependencies.
